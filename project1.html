Bridge Overload Detection System

Introduction

This system detects overload on bridges to prevent accidents by using sensors and a servo-controlled gate mechanism.

Working Principle

Two ultrasonic sensors detect the number of cars on the bridge. If 5 cars are detected, the servo gate is activated, and the buzzer sounds. When cars leave, the system resets.

Components Used

  • 2 x Ultrasonic Sensors
  • 1 x I2C LCD Display
  • 1 x Arduino UNO
  • 1 x Servo
  • 1 x Buzzer

Applications

  • Bridge safety monitoring
  • Preventing accidents due to overloading
  • Traffic management solutions
