Smart Attendance Tracker

Introduction

This system uses face recognition to automate attendance tracking for students or employees.

Working Principle

The system captures and registers faces using the ESP32-CAM module. It then recognizes the faces to mark attendance in an Excel sheet.

Components Used

  • 1 x ESP32-CAM Module

Applications

  • Automated attendance system
  • Employee or student tracking
  • Enhancing security with face recognition
