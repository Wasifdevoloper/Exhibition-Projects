Introduction

This project allows the user to control home appliances via Bluetooth using an Android app and Arduino.

Working Principle

Using the HC-05 Bluetooth module, the user can send signals to control relays that switch home appliances (e.g., lights, fans). The servo can also be controlled for other uses.

Components Used

  • 1 x HC-05 Bluetooth Module
  • 2 x One-Channel Relay Module
  • 1 x Arduino UNO
  • 1 x Buzzer
  • 1 x Servo
  • 2 x Bulbs with Bulb Holders
  • AC Power Supply

Applications

  • Home automation
  • Remote control of electrical appliances
  • Energy-saving solutions
